metcountplot {MetENP}R Documentation

counts and plots significant metabolite

Description

counts and plots significant metabolite

Usage

metcountplot(df_metclass, metclass, plotting, thres_logfC)

Arguments

df_metclass

dataframe having significant metabolite with metabolite class as obtained from map_refmet_class.R

metclass

sub_class, main_class or super_class

thres_logfC

log2 fold change threshold

plot

logical TRUE or FALSE, default true

Examples

count_changes = metcountplot(df_metclass=sig_metabolites_pubchem, metclass='sub_class', plotting=TRUE, thres_logfC = 0.5)
